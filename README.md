# 🚀 The Revolution of The Weeknd | Projeto Individual

![The Weeknd Aesthetic](https://img.shields.io/badge/Aesthetic-Starboy-red?style=for-the-badge)
![Status](https://img.shields.io/badge/Status-Em%20Desenvolvimento-black?style=for-the-badge)
![Language](https://img.shields.io/badge/Language-JavaScript-F7DF1E?style=for-the-badge&logo=javascript&logoColor=black)

Um projeto web dedicado a analisar o impacto cultural e a evolução estética de **Abel Tesfaye (The Weeknd)** na indústria musical. Este trabalho documenta como o artista redefiniu gêneros através de eras icônicas, como *Starboy*, integrando uma interface moderna com um backend funcional.

---

## 📋 Sobre o Projeto

Este é um **Projeto Individual** desenvolvido como parte da minha graduação em Ciência da Computação. O objetivo é aplicar conceitos de desenvolvimento Full Stack para criar uma experiência imersiva que explore a missão, visão e os valores que tornaram The Weeknd um fenômeno global.

### Principais Seções:
*   **Linha do Tempo de Eras:** Análise visual e textual da evolução artística.
*   **Impacto na Indústria:** Dados sobre recordes e mudanças no gênero R&B/Pop.
*   **Dashboard Interativo:** Integração com banco de dados para coletar e exibir preferências dos usuários sobre as eras do artista.

---

## 🛠️ Tecnologias Utilizadas

O projeto utiliza tecnologias fundamentais, priorizando a organização do código e as boas práticas de desenvolvimento:

*   **Frontend:**
    *   **HTML5:** Estruturação semântica.
    *   **CSS3:** Design minimalista com foco em paletas de cores escuras e contrastes vibrantes (estética *synthwave/starboy*).
    *   **JavaScript (ES6+):** Manipulação de DOM e lógica de interação. 
        *   *Nota: Uso estrito de `let` e `const` para declaração de variáveis, seguindo padrões modernos.*

*   **Backend & Banco de Dados:**
    *   **MySQL:** Modelagem de dados para armazenamento de informações dos usuários e interações com o site.
    *   **Node.js:** Para a construção da API de conexão (caso esteja utilizando).
